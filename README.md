

## Folders
* `django`- a django app
* `shop` - an angular app

## How to run 
* `cd django` - on Windows
* `pip install`
* `python manage.py makemigrations`
* `python manage.py migrate`
* `python manage.py runserver`<br/>

In the separate terminal 

* `cd shop` - on Windows
* `npm run start` 
Then open the brower on http://localhost:4200/
